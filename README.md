##Education University - Quiz Web Application
#Overview
Education University is a full-stack quiz web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).
It allows admins to create and manage quiz sets, while users can attempt quizzes and view their results instantly.
The project is designed as an interactive and efficient platform for educational institutions.

##Features

- Role-based authentication (Admin and User)

- Admin dashboard for creating and managing quizzes

- User interface for attempting quizzes and viewing results

- Real-time result and feedback system

- RESTful API integration between frontend and backend

- Responsive and user-friendly design

##Technologies Used

- Frontend: React.js, HTML, CSS, JavaScript

- Backend: Node.js, Express.js

- Database: MongoDB (Atlas)

- Version Control: Git and GitHub
